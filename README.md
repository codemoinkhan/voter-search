# Voter Search Web App

A Django-based web application for searching voter records efficiently.

## Features

- Search voters by name
- Search by voter ID
- Search by house number
- Location filtering (district, block, village, ward, booth)
- CSV voter data import
- Structured database for large voter datasets

## Tech Stack

- Python
- Django
- SQLite
- HTML / CSS / JavaScript
